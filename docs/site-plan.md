# ATLAS-V Site Plan

## Goal
Create a professional personal landing page for Pete Marino that replaces Linktree and lives on welcometomarz.net.

## Recommended first release
A fast one-page site with:
- Name and title
- Short professional bio
- Links: LinkedIn, GitHub, resume, email
- Optional social links below the fold
- Optional featured projects section

## Hosting options
1. Cloudflare Pages - recommended for custom domain and easy future growth
2. GitHub Pages - good fallback, simple and free

## Build options
1. Plain static HTML/CSS - fastest, least maintenance
2. Astro - best balance of polish and simplicity
3. Next.js static export - fine, but more machinery than this needs

## Recommendation
Start with Astro or plain static HTML.
If speed matters most, use plain static HTML.
If maintainability and nicer component structure matter, use Astro.
